# PUBG Win Prediction

## Project Overview
This project predicts the **winPlacePerc (win percentage)** in PUBG using Machine Learning models.  
The goal is to analyze player statistics and build an accurate predictive model.

---

## Dataset
The dataset contains player match statistics such as:

- kills
- damageDealt
- walkDistance
- weaponsAcquired
- heals
- boosts

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

## Machine Learning Model

Model Used:
- Random Forest Regressor

---

## Hyperparameter Tuning

Hyperparameter tuning was performed using:

- RandomizedSearchCV
- Cross Validation

This helped improve the model performance and reduce overfitting.

---

## Project Workflow

1. Data Cleaning
2. Exploratory Data Analysis (EDA)
3. Feature Engineering
4. Train Test Split
5. Model Training
6. Hyperparameter Tuning
7. Model Evaluation

---

## Model Performance

R2 Score: 0.92
